Nesse projeto elaboramos e desenvolvemos um sistema que integra uma linguagem de programação compilada (C++) com várias linguagens de scripts (Lua e Python). A ideia é fazer uma ferramenta que facilite a vida de todo programador que queira ou exportar rotinas em C++ para essas linguagens de script, ou incorporar scripts na aplicação C++ dele. Batizamos ele de Projeto Ouroboros.
